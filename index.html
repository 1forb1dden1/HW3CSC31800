Items

    This simple application allows you to manage a list of items. You can add new items to the list, view all existing items, and delete items you no longer need. The frontend is built with plain HTML, CSS, and JavaScript, and it interacts with a RESTful API created with Express.js running on a Node.js server.

    • Get Items:
      • Method: GET
      • URL:

        http://localhost:3000/api/items

    • Create Item:
      • Method: POST
      • URL:

        http://localhost:3000/api/items

      • Sample Body: 
        
                            {`"id": "1", "name": "Sample Item"`}
    • Update Item:
      • Method: PUT
      • URL:

        http://localhost:3000/api/items/1

      • Sample Body: 
        
                            {`"id": "1", "name": "Updated Item"`}
    • Delete Item:
      • Method: DELETE
      • URL:

        http://localhost:3000/api/items/1
